# Panoramica del server MCP remoto per [!DNL Customer Journey Analytics]

Scopri come collegare strumenti di intelligenza artificiale come [!DNL Claude], [!DNL ChatGPT] o [!DNL Cursor] al server MCP di Adobe Customer Journey Analytics (CJA) per esplorare le visualizzazioni dati, eseguire rapporti e generare informazioni sui KPI pronte per l’esecutivo utilizzando il linguaggio naturale. Questa demo mostra come i server MCP remoti abilitano flussi di lavoro di analisi scalabili e gestiti e rapporti automatizzati in Adobe Experience Platform.

Per ulteriori informazioni, rivedi la [documentazione](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"} di Content Analytics.
