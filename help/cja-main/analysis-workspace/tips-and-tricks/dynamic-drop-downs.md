# Creare segmenti dinamici a discesa in Analysis Workspace

Scopri come creare segmenti a discesa basati su un’intera dimensione. Gli elementi di Dimension vengono aggiornati in modo dinamico in base all’intervallo di date utilizzato nel pannello del rapporto. Per ulteriori informazioni, consulta la [documentazione](https://experienceleague.adobe.com/it/docs/analytics-platform/using/cja-components/cja-segments/create-filters){target="_blank"}.
