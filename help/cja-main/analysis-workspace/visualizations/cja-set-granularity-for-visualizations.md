# Impostare la granularità per le visualizzazioni

Regola facilmente la modalità di visualizzazione dei dati utilizzando il selettore di granularità nelle impostazioni di visualizzazione delle tendenze. Questa funzione semplifica l’analisi dei dati consentendo di perfezionare le visualizzazioni in base alle esigenze. Scopri come ottimizzare i flussi di lavoro e rendere più accessibili le informazioni sui dati.
